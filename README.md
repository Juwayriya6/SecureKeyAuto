# Secure Key Auto Ltd — Premium Static Website

A GitHub-ready static website for Secure Key Auto Ltd using the orange and blue brand identity.

## Included

- Premium responsive homepage
- Orange + blue brand styling
- Animated hero section
- Floating emergency CTA
- Mobile navigation
- Service pages
- Location SEO pages
- FAQ accordion
- Contact/quote forms using email fallback
- Google Maps embed placeholder
- Simple AI-style chatbot widget
- SEO meta tags
- LocalBusiness schema markup
- Dark mode support based on system preference

## How to deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this ZIP.
3. Go to repository **Settings > Pages**.
4. Set source to your main branch and root folder.
5. Save and wait for GitHub Pages to publish.

## How to deploy on Vercel

1. Create a new GitHub repository.
2. Upload all files from this ZIP.
3. Import the repository into Vercel.
4. Add your domain: `securekeyauto.com`.
5. Deploy.

## Before Launch

Update these placeholders:

- Phone number: search `+10000000000`
- Email: search `service@securekeyauto.com`
- Service areas in `/locations`
- Google Maps embed
- Reviews
- Business address if applicable
- Form backend, if you want real form submissions instead of email draft
